Restarting

The first week’s project presented a series of challenges, logistical and technical. I have a Windows laptop, which, from what I understand, is more difficult to build a website on using Bracket's program. I had to transition to a Mac Desktop computer for this project (a computer I am lucky enough to have access to through using either ther Fab Lab or the Fiber Space). Getting used to the Mac set up in it of itself was a process. I have the amazing TA’s and employees (wise technology sages) at the MakerSpace to thank for being patient teachers while helping a novice web designer troubleshoot the headaches of a first website.

Ran into a couple roadblocks...

When I tried to get to my website a full 24 hours after I had committed it, the changes still had not loaded and the 404 page came up again. I watched Madison’s instructions again and consulted the TA in the MakerSpace and went to my github repository to see if my link was incorrect. I found that I had not used my username for my website which was why it was not working. At this point I also switched computers from the Mac in the Fiberspace to the Mac in the MakerSpace. Thus, I had to essentially redo everything I had done before under my new and correct website name, emilygray22[dot]github[dot]io.

By “redo everything,” I mean pretty much just get the same template onto Brackets and start from where I was.

Getting down to business...

I started to rename the tabs in the nav bar, beginning with an “About” page. I linked this new file under the website and put “about.html” in my index nav bar where the filler “#” used to be so it would actually lead to my new About page.

I committed these changes and pushed them, to see how this update would manifest if I searched up my link.

I then started to personalize my about page. I uploaded an image for my home/index page and edited the content so that it would reflect my desired usability. I deleted a couple tabs I did not need from the nav bar and made sure each new tab I added successfully linked from one to another.

I created my bio for my About page and uploaded a picture to go along with it. The picture was massive when I first uploaded it. I was able to resize it by adjusting the width percent.

A huge moment for me was getting the images up on the website. Being able to see my art blown up for a website was so gratifying. That was a turning point in my moral for sure. Little formatting problems that, previously I would have run from or just not tried to fix for fear of messing everything up, I was able to work through and fix. This was an incredibly satisfying project. Entry into the technical world to which I had kept myself shunned from for so long, has shown me how artistic creativity is implicit in fields to which I had thought it was foregin.
